Almd = Mac: ⌘/Cmd key; Win/Linux: Alt key

Most Essential Editor Commands:
KEY:UP:NAV Up
KEY:DOWN:NAV Down
KEY:LEFT:NAV Left
KEY:RIGHT:NAV Right
*KEY:HOME:NAV Home
KEY:OSX:END:NAV Fn-Right
KEY:WLX:END:NAV End
*KEY:BEGINLINE:NAV
KEY:OSX:HOMETOGGLE:NAV Fn-Left
KEY:WLX:HOMETOGGLE:NAV Home
KEY:GOTOLINE:NAV Ctrl-g
KEY:OSX:PGUP:NAV Fn-Up
KEY:WLX:PGUP:NAV PageUp
KEY:OSX:PGDN:NAV Fn-Down
KEY:WLX:PGDN:NAV PageDown
KEY:OSX:TOP:NAV Cmd-Up
KEY:WLX:TOP:NAV Ctrl-Home
KEY:OSX:BOTTOM:NAV Cmd-Down
KEY:WLX:BOTTOM:NAV Ctrl-End
*KEY:JUMPPAREN:NAV Ctrl-Shift-\

*KEY:SAVEFILE:NAV Ctrl-s
*KEY:SAVEAS:NAV Ctrl-Shift-s
*KEY:SAVEALL:NAV Ctrl-k s
*KEY:CLOSEFILE:NAV Win-Alt f c
*KEY:CLOSEWIN:NAV Ctrl-k w
*KEY:CLOSEAPP:NAV Win-Alt f x
*KEY:TREETOGGLE:NAV Ctrl-b
*KEY:TREEFOCUS:NAV Ctrl-Shift-e
*KEY:NEXTWIN:NAV
*KEY:PREVWIN:NAV
*KEY:GOTOWIN:NAV Ctrl-1 etc go to numbered editor window
*KEY:NEXTBUFFER:NAV Ctrl-Tab or Ctrl-k Ctrl-PageDown
*KEY:PREVBUFFER:NAV Ctrl-Shift-Tab or Ctrl-k Ctrl-PageUp
*KEY:MOVEWIN:NAV Ctrl-k Arrow
*KEY:OPENANY:NAV Ctrl-p
*KEY:PREVIEW:NAV Ctrl-k v or Ctrl-Shift-v
*KEY:REOPENWIN:NAV Ctrl-Shift-t reopen closed editor tab

*KEY:SPLITLEFT:LYT
*KEY:SPLITRIGHT:LYT Ctrl-\
*KEY:SPLITUP:LYT Ctrl-\ Ctrl-k Up
*KEY:SPLITDOWN:LYT Ctrl-\ Ctrl-k Down
*KEY:ONEWIN:LYT

*KEY:INCSEARCH:FIND
*KEY:FIND:FIND Ctrl-f
*KEY:NEXTMATCH:FIND F3 or Enter
*KEY:PREVMATCH:FIND Shift-F3 or Shift-Enter
*KEY:JUMPDEF:CODE Win-Alt g d Enter
*KEY:JUMPREF:CODE Ctrl-F12
*KEY:JUMPBACK:FIND Alt-Left

KEY:OSX:SELECTALL:MARK Cmd-Up Shift-Cmd-Down
*KEY:SELECTALL:MARK Ctrl-a
*KEY:SELECTLINE:MARK Ctrl-l or Shift-Down
*KEY:SELECTPAREN:MARK Alt-Shift-Right  (expand selection until good)
KEY:SELECTVBLOCK:MARK Shift-Alt-MouseDrag
KEY:SELECTVBLOCK:MARK Shift-Arrow
KEY:TOGGLESELECTVBLOCK:MARK command palette Toggle Column Selection Mode
KEY:CURSORADD:CURS Almd-MouseClick
KEY:OSX:CURSORADDABOVE:CURS Shift-Up
KEY:WLX:CURSORADDABOVE:CURS Shift-Alt-Up
KEY:OSX:CURSORADDBELOW:CURS Shift-Down
KEY:WLX:CURSORADDBELOW:CURS Shift-Alt-Down
KEY:CURSORADDMARKED:CURS Ctmd-Shift-l

*KEY:COPY:PASTE Ctrl-c
*KEY:CUT:PASTE Ctrl-x
*KEY:PASTE:PASTE Ctrl-v
KEY:COPYLINE:PASTE Ctmd-c (when nothing selected)
KEY:CUTLINE:PASTE Ctmd-x
KEY:WLX:DUPLINE:PASTE Ctrl-c Ctrl-v (when nothing selected)
KEY:OSX:DUPLINE:PASTE Alt-Shift-Down
KEY:OSX:DUPLINEUP:PASTE Alt-Shift-Up
*KEY:VBLOCK:PASTE
*KEY:VBLOCKFILL:PASTE

*KEY:LINECOMMENT:EDIT Win-/
KEY:DELLINE:EDIT Ctmd-Shift-k
*KEY:DELENDLINE:EDIT Shift-End Del
*KEY:BSBEGINLINE:EDIT Shift-Home Del
*KEY:SPLITLINE:EDIT Enter
*KEY:JOINLINE:EDIT command palette join
*KEY:OPENLINEBELOW:EDIT Ctrl-Enter
*KEY:OPENLINEUP:EDIT Ctrl-Shift-Enter
KEY:MOVEUP:EDIT Alt-Up
KEY:MOVEDN:EDIT Alt-Down
*KEY:INDENT:EDIT Ctrl-]
*KEY:OUTDENT:EDIT Ctrl-[
*KEY:REPLACE:EDIT Ctrl-h
*KEY:REPLACESEL:EDIT
*KEY:UNDO:EDIT Ctrl-z
*KEY:REDO:EDIT Ctrl-y
*KEY:CASECHANGE:EDIT command palette case
*KEY:TOGGLEQUOTE:EDIT
*KEY:WRAPQUOTE:EDIT
*KEY:WRAPPAREN:EDIT

*KEY:FONTBIGGER:VIEW Ctrl-=
*KEY:FONTSMALLER:VIEW Ctrl-Minus
*KEY:ZENVIEW:VIEW Ctrl-k z Mostly just the editor view
*KEY:NOZENVIEW:VIEW Esc Esc Back before zen mode

KEY:COMMAND:CMD F1 or Ctmd-Shift-p
KEY:SHELL:CMD Ctrl-`
*KEY:BINDKEY:CMD Ctrl-k Ctrl-s
*KEY:SETTINGS:CMD Ctrl-,

KEY:COMPLETE:CODE Ctrl-Space
