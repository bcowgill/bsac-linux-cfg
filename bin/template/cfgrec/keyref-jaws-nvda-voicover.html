JAWS COMMANDS (IE) NVDA COMMANDS (FF/GC) VOICE OVER COMMANDS (MAC)  Used To
Browser Internet Explorer Firefox, Google Chrome   Close all other browsers/windows when screen reading
Screen Reader Key INSERT = INS+ INSERT = INS+ CTRL+ALT = VO+ Can be configured, but INSERT is the default
Activate Screen Reader CTRL+Alt+j CTRL+ALT+n CMD+F5 Assign hot key to Jaws by right clicking icon/properties/shortcut
Exit Screen Reader INS+F4 <ENTER> INS+q <ENTER> CMD+F5 Close the screen reader immediately
Screen Reader Menu INS+j INS+n   Show the screen reader menu to access tools, options.
  INS+F2     Jaws Manager menu
Key Help INS+SPACE j INS+n h q   Search/Help on Screen Reader Key Commands
Say Key Help   INS+1   Report key press and Screen Reader command associated
Speech View Window INS+SPACE h INS+n t s   Open window which shows speech history or continually
Pass Key Through INS+3 INS+F2   Speech Reader ignore next key let windows have it.
Shush CTRL CTRL CTRL Stop current speech
Shut up!   INS+SHIFT+S   Don't read in current application (toggle)
Refresh   INS+F5   Rescan the page for accessibility (after a dialog opens, etc)
         
Voice Settings INS+j o v v INS+n p v   Open dialog to change voice, rate, etc
Next Voice Setting   CTRL-INS-RIGHT VO+CMD+RIGHT Move through voice control settings to adjust up/dn
Faster / More CTRL-ALT-PGUP CTRL-INS-UP VO+CMD+UP Voice rate/setting faster
Slower / Less CTRL-ALT-PGDN CTRL-INS-DOWN VO+CMD+DOWN Voice rate/setting slower
Say line INS+UP INS+UP VO+s Say the current line
Say all INS+DOWN INS+DOWN VO+a Say everything from this point on.
  LEFT     Rewind during say all
  RIGHT     Skip forward during say all
         
Page Help INS+F1     List of commands available on this page type
Structure List INS+F3 INS+F7 VO+u List of page elements/structure (links, headers, landmarks, etc)
Word List CTRL+INS+w     List of all words in the page
Headings List INS+F6        (h) INS+F7 VO+u RIGHTx2 List of all headings in the page
Links List INS+F7 INS+F7  (k) VO+u RIGHT List of all links in the page
Form Elements List INS+F5 (f) VO+u RIGHTx3 List of all form elements in the page
EditBoxes List CTRL+INS+e (e)   List of all edit boxes in the page
Combo Boxes List CTRL+INS+c (c)   List of all combo boxes in the page
Radio Buttons List CTRL+INS+a    (a) (r)   List of all radio buttons in the page
Check Boxes List CTRL+INS+x (x)   List of all check boxes in the page
Buttons List CTRL+INS+b (b)   List of all buttons in the page
Tables List CTRL+INS+t (t)   List of all tables in the page
Single Letter Navigation   INS+SHIFT+SPACE   Toggle single letter navigation on/off
Next of type h f e c a x b t h f e c r x b t   Move to next of type (h=heading) + SHIFT for previous of type
Next heading t (SHIFT+T) t (SHIFT+T) VO+CMD+h (VO+CMD+SHIFT+H) Move to next / previous heading
Next link   k (SHIFT+K) VO+CMD+l (VO+CMD+SHIFT+L) Move to next / previous link
Next unvisited link u (SHIFT+U) u (SHIFT+U)   Move to next / previous unvisited link
Next visited link v (SHIFT+V) v (SHIFT+V) VO+CMD+v (VO+CMD+SHIFT+V) Move to next / previous visited link
Next frame m (SHIFT+M) m (SHIFT+M)   Move to next / previous frame
Go to main region q (SHIFT+Q)     Go to main content region
Next region r (SHIFT+R) d (SHIFT+D)   Move to next / previous region/landmark
Next heading level 1…6 (SHIFT-1…6) 1…6 (SHIFT-1…6)   Move to next / previous heading of level 1…6
Next table t (SHIFT+T) t (SHIFT+T) VO+CMD+t (VO+CMD+SHIFT+T) Move to next / previous table
Next list l (SHIFT+L) l (SHIFT+L) VO+CMD+x (VO+CMD+SHIFT+X) Move to next / previous list
Next list item i (SHIFT+I) i (SHIFT+I)   Move to next / previous list item
Next article o (SHIFT+O)     Move to next / previous article
Next annotation   a (SHIFT+A)   Move to next / previous annotation
Next place marker k (SHIFT+K)     Move to next / previous place marker
Next blockquote   q (SHIFT+Q)   Move to next / previous blockquote
Next object   o (SHIFT+O)   Move to next / previous embedded object
Next separator   s (SHIFT+S)   Move to next / previous separator
Next graphic g (SHIFT+G) g (SHIFT+G) VO+CMD+g (VO+CMD+SHIFT+G) Move to next / previous graphic
Next text   n (SHIFT+N)   Move to next / previous unlinked text
Next of same s (SHIFT+S)     Move to next / previous of same type
Next of different d (SHIFT+D)     Move to next / previous of different type
Next span y (SHIFT+Y)     Move to next / previous span
         
Next Tab stop TAB (SHIFT+TAB) TAB (SHIFT+TAB) TAB (SHIFT+TAB) Go to next / previous focusable item (link, button, input, etc)
Use Link ENTER ENTER ENTER or VO+SPACE Activate link
Use Button ENTER or SPACE ENTER or SPACE ENTER or SPACE or VO+SPACE Activate button
Navigate list ARROWS ARROWS   Navigate select lists and other aria controls
Say next line DOWN (UP) DOWN (UP) VO+RIGHT (VO+LEFT) Say the next / previous item/line
Say next letter RIGHT (LEFT) RIGHT (LEFT)   Say the next / previous letter
Next table column CTRL+ALT+RIGHT CTRL+ALT+RIGHT RIGHT or VO+RIGHT Go to the next column in current table row
Previous table column CTRL+ALT+LEFT CTRL+ALT+LEFT LEFT or VO+LEFT Go to the previous column in current table row
Next table row CTRL+ALT+DOWN CTRL+ALT+DOWN DOWN or VO+DOWN Go to next row in current table column
Previous table row CTRL+ALT+UP CTRL+ALT+UP UP or VO+UP Go to previous row in current table column
