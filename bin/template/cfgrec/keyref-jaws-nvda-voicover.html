JAWS COMMANDS (IE) NVDA COMMANDS (FF/GC) VOICE OVER COMMANDS (MAC)  Used To    
Browser Internet Explorer Firefox, Google Chrome Any Close all other browsers/windows when screen reading    
Screen Reader Key INSERT = INS+ INSERT = INS+ CTRL+ALT = VO+ Can be configured, but INSERT is the default    
Activate Screen Reader CTRL+Alt+j CTRL+ALT+n CMD+F5 Assign hot key to Jaws by right clicking icon/properties/shortcut    
Exit Screen Reader INS+F4 <ENTER> INS+q <ENTER> CMD+F5 Close the screen reader immediately    
Screen Reader Menu INS+j INS+n VO+FN+8 Show the screen reader menu to access tools, options.    
Other menus INS+F2   VO+d  VO+m  VO+mm Jaws Manager menu, Mac dock, app menu, app status    
Key Help INS+SPACE j INS+n h q VO+hh Search/Help on Screen Reader Key Commands    
Say Key Help   INS+1 VO+k       (ESC) Report key press and Screen Reader command associated    
Speech View Window INS+SPACE h INS+n t s VO+CMD+F10 VO+SHIFT+F10 Open/hide/resize window which shows speech history or continually    
Pass Key Through INS+3 INS+F2 VO+TAB Speech Reader ignore next key let windows have it.    
Shush CTRL CTRL CTRL Stop current speech    
Shut up!   INS+SHIFT+S   Don't read in current application (toggle)    
Refresh   INS+F5   Rescan the page for accessibility (after a dialog opens, etc)    
             
Voice Settings INS+j o v v INS+n p v   Open dialog to change voice, rate, etc    
Next Voice Setting   CTRL-INS-RIGHT VO+CMD+RIGHT Move through voice control settings to adjust up/dn    
Faster / More CTRL-ALT-PGUP CTRL-INS-UP VO+CMD+UP Voice rate/setting faster    
Slower / Less CTRL-ALT-PGDN CTRL-INS-DOWN VO+CMD+DOWN Voice rate/setting slower    
Say line INS+UP INS+UP VO+s Say the current line    
Say all INS+DOWN INS+DOWN VO+a Say everything from this point on.    
  LEFT     Rewind during say all    
  RIGHT     Skip forward during say all    
             
Page Help INS+F1     List of commands available on this page type    
Structure List INS+F3 INS+F7 VO+u List of page elements/structure (links, headers, landmarks, etc)    
Word List CTRL+INS+w     List of all words in the page    
Headings List INS+F6        (h) INS+F7 VO+u RIGHTx2 List of all headings in the page    
Links List INS+F7 INS+F7  (k) VO+u RIGHT List of all links in the page    
Form Elements List INS+F5 (f) VO+u RIGHTx3 List of all form elements in the page    
EditBoxes List CTRL+INS+e (e)   List of all edit boxes in the page    
Combo Boxes List CTRL+INS+c (c)   List of all combo boxes in the page    
Radio Buttons List CTRL+INS+a    (a) (r)   List of all radio buttons in the page    
Check Boxes List CTRL+INS+x (x)   List of all check boxes in the page    
Buttons List CTRL+INS+b (b)   List of all buttons in the page    
Tables List CTRL+INS+t (t) VO+u RIGHTx5 List of all tables in the page    
Single Letter Navigation   INS+SHIFT+SPACE   Toggle single letter navigation on/off
