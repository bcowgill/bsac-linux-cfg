Unix / Linux pico/nano editor reference

General notes:
Ctrl-x = Esc Esc x
Meta-x = Alt-x = Esc x
Esc Esc nnn = insert ASCII character nnn

Additional cmmands of interest in nano:

KEY:INSERTFILE:EDIT C-r or F5 Insert a named file into the current one.
KEY:CURSORPOS:NAV C-c or F11 Show cursor position row/col/chars and %
KEY:INSERTCHAR:EDIT M-v Insert the next character verbatim
KEY:INSERTTAB:EDIT C-i Insert a hard tab character
KEY:INSERTCR:EDIT C-m Insert a hard newline character
KEY:REFRESH:CMD C-l Redraw/refresh the current screen
