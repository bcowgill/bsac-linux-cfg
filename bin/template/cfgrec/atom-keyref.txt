atom editor command quick reference

http://flight-manual.atom.io/getting-started/sections/atom-basics/#platform-linux

*** marks my most used commands -- needed in any other emulation

KEY:ATOM:CMD Ctrl-Shift-p Open the command palette to execute a command ***
KEY:ATOM:CMD Ctrl-q Exit atom
KEY:ATOM:CONFIG Ctrl-, Open the settings tab
KEY:ATOM:NAV Alt-\ Focus to the Tree View
KEY:ATOM:NAV Ctrl-Shift-\ Show active file in Tree View
KEY:ATOM:VIEW Ctrl-\ Toggle the Tree View ***
KEY:ATOM:VIEW Ctrl-x Close current tab
KEY:ATOM:VIEW Ctrl-F4 Close current tab
KEY:ATOM:VIEW Ctrl-Shift-9 Toggle git tab
KEY:ATOM:VIEW Ctrl-9 Focus to git tab
KEY:ATOM:VIEW Ctrl-Shift-8 Toggle github tab
KEY:ATOM:VIEW Ctrl-8 Focus to github tab
KEY:ATOM:VIEW Ctrl-Shift-m Toggle Markdown preview window ***
KEY:ATOM:VIEW Ctrl-k left Split pane to left
KEY:ATOM:VIEW Ctrl-k right Split pane to right
KEY:ATOM:VIEW Ctrl-k up Split pane above
KEY:ATOM:VIEW Ctrl-k down Split pane below
KEY:ATOM:CURSOR Ctrl-MouseClick Add/Remove a multiple cursor location to allow inserting same text in multiple locations.
KEY:ATOM:CURSOR Esc Clear all multiple cursor locations
KEY:ATOM:EDIT Cmd-j [Edit/Lines/Join] Join two lines removing unnecessary space
KEY:ATOM:EDIT Cmd-/ Line Comment/Uncomment the current line or selected lines ***
KEY:ATOM:EDIT Ctrl-Shift-p Change   Change case commands ***
KEY:ATOM:EDIT Ctrl-up Move current line up one line ***
KEY:ATOM:EDIT Ctrl-down Move current line down one line ***
KEY:ATOM:MARK Ctrl-Alt-m Mark text between matching brackets ***
KEY:ATOM:FIND Ctrl-f Find/Replace text
KEY:ATOM:FIND Ctrl-Shift-f Find/Replace text in project
KEY:ATOM:FIND Ctrl-m Move to matching bracket ***
KEY:ATOM:CODE Ctrl-Shift-r View symbols/methods in project (needs a tag file)
KEY:ATOM:FIND Ctrl-t Fuzzy Finder - popup to open any file by incremental search ***
KEY:ATOM:CODE Ctrl-r View symbols/methods and jump to definition
KEY:ATOM:CODE Ctrl-Alt-down Go to symbol declaration (needs a tag file)
KEY:ATOM:CODE Ctrl-Alt-up Jump back from symbol declaration
KEY:ATOM:FUN Ctrl-Alt-o Toggle activation of power mode

Package keymaps:
keep-case-replace uses Ctrl-Cmd-r
string-case uses
  C-k C-c camelCase
  C-k C-k kebab-case
  C-k C-p PascalCase
  C-k C-s snake_case
          CONST_CASE
change-case uses
  C-k C-c camelCase
  C-k C-s snake_case
atom-change-case-menu has no hotkeys but many change modes.

duplicate-line-or-selection uses
	C-S-d or Cmd-S-d to duplicate the line or selection
highlight-selected uses
	C-Cmd-h to toggle the highlighte matches
related uses
	C-S-r to open a related file
sort-lines use command palette search for Sort
	F5 default sort lines
sort-css
	F6 sort selected lines by CSS order
Sublime-Style-Column-Selection
	Alt-MouseDrag to select text by blocks on Windows/Mac
	Shift-MouseDrag to select text by blocks on Linux
cursor-index uses
	Alt-g to go to an exact cursor index location
toggle-quotes uses
	C-" or Cmd-" to toggle between quote types
atom-wrap-in-tag uses
	C-S-w to wrap selected text in an HTML tag
autocomplete uses
	C-Space to offer autocompleteion suggestions
autocomplete-date uses macros to insert dates
	now date (dd) time (dt)
color-picker uses
	C-A-c or Cmd-S-c or right click to pick a color
Chromo-Color-Previews uses
	C-A-o to toggle color preview gutter on/off
nms-color-bracket uses
	A-q to toggle coloring different matching braces
color-whitespace uses
	C-A-c to control coloring indentation by control flow types
color-tabs uses
	C-A-a to assign a random color to current tab
	C-A-x to remove color from current tab
line-number-color uses
	C-A-o to toggle line number color
