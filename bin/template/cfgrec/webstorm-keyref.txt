Webstorm editor command quick reference

KEY:WEBSTORM:NAV Alt-Shift-1 Quickly find the currently edited file in another view (Project/Structure etc)
KEY:WEBSTORM:NAV Escape in a tool window gets you to the editor
KEY:WEBSTORM:NAV Shift-Escape in a tool window gets you to the editor and hides the tool window
KEY:WEBSTORM:NAV F12 move from the editor to the last focussed tool window

KEY:WEBSTORM:VIEW Shift-mousewheel horizontal scrolling of the window

KEY:WEBSTORM:FIND Ctrl-Shift-N [Navigate/File] popup to open any file by incremental search
KEY:WEBSTORM:FIND Ctrl-n [Navigate/Class] popup to find any class by incremental search use @class markers to mark your classes
KEY:WEBSTORM:FIND Ctrl-Alt-Shift-n [Navigate/Class] popup to find any symbol by incremental search
KEY:WEBSTORM:FIND Alt-Shift-7 [RightClick/Find Usages] find usages of symbol at cursor show in the find window
KEY:WEBSTORM:FIND Ctrl-b jump to declaration of a class, method or variable at the cursor
KEY:WEBSTORM:FIND Ctrl MouseClick on symbol jump to declaration of a class, method or variable clicked
KEY:WEBSTORM:FIND Ctrl-0 [Navigate/File Structure] show members of the current class and allow to jump straight to them incremental search

KEY:WEBSTORM:CODE Ctrl-Space code completion (html attributes, object properties, etc)
KEY:WEBSTORM:CODE Tab insert selected code completion name and erase anything to right of cursor
KEY:WEBSTORM:CODE Enter insert selected code completion name keeping anything to right of cursor
KEY:WEBSTORM:CODE Shift-F6 [Refactor/Rename] rename class/methods/variables with automatic correction of all references.

KEY:WEBSTORM:DOC Ctrl-q [View/Quick Documentation] show @doc documentation for symbol at cursor
