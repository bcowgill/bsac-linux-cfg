Emacs Key Binding Reference

Leaving Emacs

Ctrl-zsuspend Emacs (or iconify it under X)
Ctrl-x Ctrl-cexit Emacs ask to save

Error Recovery

Ctrl-gabort a partially typed or executing command
Alt-x recover-sessionrecover files lost by a system crash
Ctrl-/ or Ctrl-x u or Ctrl-_undo an unwanted change
Alt-x revert-bufferrevert a buffer to what is in the disk file
Alt-x redraw-displayredraw screen

Modifier Keys

Alt-x function-nameExecutes a named command function
Ctrl-u number/-numberSpecify a +/- numeric argument for the next command
Alt-NSpecify a numeric argument for the next command
Alt-- or Ctrl-- or Ctrl-Alt--Begin a negative argument for the next command
Ctrl-q charInsert the next character literally

Other Motion

Ctrl-lscroll current line to center, top or bottom in succession
Ctrl-x <scroll view to left
Ctrl-x >scroll view to right
Ctrl-Alt-bbackwards one symbolic expression (sexp)
Ctrl-Alt-fforwards one symbolic expression (sexp)
Alt-g ggo to line number
Alt-g cgo to character offset from start of document
Alt-mgo to start of line skipping initial indentation

Marking (Selecting)

Ctrl-Space or Ctrl-@set mark here (begin selecting)
Ctrl-x Ctrl-xexchange mark and point
Alt-@set mark N words away (with argument prefix combo)
Ctrl-Alt-@mark forward within curent symbolic expression
Ctrl-Alt-hmark current function
Alt-hmark current paragraph
Ctrl-x Ctrl-pmark current page
Ctrl-x hmark entire buffer

Copy, Cut, Paste

Ctrl-wkill (cut) marked region to kill ring
Alt-wcopy marked region to kill ring
Ctrl-yyank (paste) back from kill ring
Alt-yyank (paste) previous entry from kill ring
Alt-z charkill through next occurrence of char
Alt-- Ctrl-Alt-kkill backwards one symbolic expression (sexp)
Ctrl-Alt-kkill forwards one symbolic expression (sexp)

Files and Buffers

Ctrl-x Ctrl-fvisit a file into a buffer (existing or virtual new)
Ctrl-x Ctrl-ssave the buffer to its disk file (create if virtual)
Ctrl-x Ctrl-wsave buffer as another name
Ctrl-x ssave all files asking to save each
Ctrl-x iinsert contents of another file into this buffer
Ctrl-x Ctrl-vreplace this buffer with the file you really want
Ctrl-x Ctrl-qtoggle read-only status of buffer

Getting Help

Ctrl-h or F1help prefix key
Ctrl-h thelp tutorial
Ctrl-Alt-vscroll the help (other) window
Ctrl-x 1remove help (and all other) window
Ctrl-h aapropos help matching a string
Ctrl-h cname of function bound to a key
Ctrl-h khelp on the function bound to a key
Ctrl-h fhelp on a named function
Ctrl-h mhelp on current editing mode

Incremental Search

Ctrl-ssearch forward
Ctrl-rsearch backward
Ctrl-Alt-sregular expression search forward
Ctrl-Alt-rregular expression search backward
Alt-pselect previous search string from history
Alt-nselect next search string from history
Enterexit incremental search
Backspaceundo effect of last character
Ctrl-gabort current search

Search and Replace

Alt-%begin literal search and replace
Ctrl-Alt-%begin regex search and replace
Ctrl-q Ctrl-jsearch or replace a literal newline
Space or yreplace and find next
,replace and stay here
Ctrl-renter an edit mode to custom replace this match
Ctrl-Alt-cfinish custom replace and carry on with replacement
Backspace or nskip to next without replacing
!replace all remaining matches
^back up to the previous match
Enterexit search and replace
Ctrl-hget help while replacing
